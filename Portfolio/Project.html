Mastermind : A Console Application

This mastermind game was a 1st year university project that was aimed at developing our knowledge of algorithms. It is a simple console-based application that was to be written on one page so we could focus on the pure code rather than object-oriented programming.

In order to play the game, the user selects the length of the number code they want to guess as well as the maximum value for each digit in the code. The aim is to guess the secret code with multiple attempts, and each guess has white and black pieces that indicate how correct the guess is.
